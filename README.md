
## flameshot-ocr

为了截图取文字，保留flameshot所有功能，在原基础上加上ocr功能，使用比较方便。

## 编译方式

参考flameshot

## OCR

目前仅支持百度ocr，白漂500次一个月

#### 配制ocr

```json
{
        "baidu":{
                "app_id":"你的app_id",
                "api_key":"你的app_key",
                "secret_key":"你的secret_key"
        }
}
```

参考[百度OCR](https://cloud.baidu.com/doc/OCR/s/Kk3h7y7vq)
